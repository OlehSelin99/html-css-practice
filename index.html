Introduction

To create websites, you need to know about HTML (HyperText Markup Language) - a fundamental technology that is used to define the structure of a web page.

HTML is used to define how your content should be displayed in different forms:

  • Paragraph
  • List
  • Heading
  • Link
  • Image
  • Media
  • Forms
There are many other elements available, and you can also create your own new element.

Topics to learn

To create your first page with HTML, you need to understand some topics:
  1. Working with text
  2. Attributes
  3. Links
  4. Images
  5. Tables
  6. Forms
